6 ## 1. Description
7 
8 We define numerical methods that are based on discretizations preserving the
9 properties of their continuum counterparts to be **mimetic**.
10 
11 The **Mimetic Methods Toolkit (MTK)** is a C++ library for mimetic numerical
12 methods. It is arranged as a set of classes for **mimetic quadratures**,
13 **mimetic interpolation**, and **mimetic discretization** methods for the
14 numerical solution of ordinary and partial differential equations.
15 
16 This collection of **MATLAB Wrappers for the MTK (MMTK)** allows developers to
17 invoke the MTK from a MATLAB environment.
18 
19 Finally, a collection of grid visualization routines that is compatible with
20 these wrappers can be found in:
21 
22 [MATLAB Visualizers for Uniform Staggered Grids](https://github.com/ejspeiro/UniStgGrid-Visualizers)

47 ## 3. Installation
48 
49 The following instructions assume MATLAB R2014a or greater.
50 
51 You have two options, either follow the instructions given, **expert install**,
52 or use the provided **patch file**, **naive install**.

55 
56 From your MATLAB prompt, type:
57 
58 ```

71 ### EXPERT PART 2. CONFIGURATION OF THE MEX COMPILER SETUP FILE.
72 
73 The previous step creates the following file:
74 
75 ```
76 $(HOME)/.matlab/R2014a/mex_C++_glnxa64.xml
77 ```
78 
79 The purpose of this section is to configure the MATLAB R2014a Mex compiler so
80 that it can work with the latest C++ standard (C++11).
81 
82 Please execute the following changes on the aforementioned file (line numbers
83 may differ):
84 
85 Lines 26 and 27:
86 
87 ```
88 CMDLINE1="$CXX -std=c++11 -c $DEFINES $INCLUDE $CXXFLAGS $OPTIM $SRC -o $OBJ"
89 CMDLINE2="$LDXX -std=c++11 $LDFLAGS $LDTYPE $LINKOPTIM $LINKEXPORT $OBJS $CXXLIBS $LINKLIBS -o $EXE"
90 ```
91 
92 Line 33:
93 
94 ```
95 CXXFLAGS="-std=c++11 -ansi -fexceptions -fPIC -fno-omit-frame-pointer -pthread"
96 ```
97 
98 Line 35 and 26:
99 
100 ```
101 CXXOPTIMFLAGS="-std=c++11 -O -DNDEBUG"
102 CXXDEBUGFLAGS="-std=c++11 -g"
103 ```

106 
107 ```

134 ### NAIVE PART 2. CONFIGURATION OF THE MEX COMPILER SETUP FILE.
135 
136 ```
137 cd $HOME/.matlab/R2014a
138 chmod +w mex_C++_glnxa64.xml
139 patch < mex_C++_glnxa64.patch
140 chmod -w mex_C++_glnxa64.xml
141 ```

144 ### PART 3: CONFIGURATION OF THE MAKEFILE.
145 
146 The following steps are required the build and test the MTK. Please use the
147 accompanying `Makefile.inc` file, which should provide a solid template to
148 start with. The following command provides help on the options for make:
149 
150 ```
151 $ make help
152 -----
153 Makefile for the MMTK.
154 
155 Options are:
156 - all: builds he library, the tests, and examples.
157 
158 - gendoc: generates the documentation for the library.
159 
160 - clean: cleans ALL the generated files.
161 -----
162 ```
163
